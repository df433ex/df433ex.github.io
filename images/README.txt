Drop your images in this folder using the exact filenames below.
The portfolio page is wired to look for them automatically.
If a file is missing, the colored gradient fallback shows instead.

  portrait.jpg       — your headshot for the About Me section (4:5 ratio works best)
  azzurri.jpg        — The Azzurri Decline card (16:10 ratio)
  breadbutter.jpg    — Bread & Butter Market Audit card (16:10)
  gametheory.jpg     — Game Theory project card (16:10)
  citi.jpg           — Forage / Citi quant project card (16:10)

JPEG or PNG both work. Recommended sizes: ~1200x750 px for project thumbs,
~1000x1250 px for the portrait. Bigger is fine — they get scaled down.
